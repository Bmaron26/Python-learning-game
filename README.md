# Python Quest - Learn Python RPG

A browser-based RPG adventure game that teaches Python through combat encounters. **No installation needed** — just open the link on your phone or computer and start playing!

## Play Now

**https://bmaron26.github.io/Python-learning-game/**

Just tap the link above on any device — phone, tablet, or computer.

## Features

- **RPG Adventure** — Battle enemies across 5 chapters with Python knowledge as your weapon
- **13 Quests** — Fight goblins, dragons, and a final boss with increasingly difficult Python questions
- **Character Progression** — Level up, earn XP and gold, collect items
- **Mobile-Friendly** — Designed for phones first, works on any screen size
- **Save Progress** — Your game auto-saves in the browser so you can pick up where you left off
- **No Install Required** — Pure HTML/CSS/JS, runs entirely in the browser

## Game Overview

### Chapters
1. **The Village of Variables** — Learn Python basics (variables, types)
2. **The Forest of Flow** — Master control flow (if/elif, loops, range)
3. **The Cavern of Collections** — Lists, dicts, and tuples
4. **The Tower of Functions** — Functions, lambdas, and return values
5. **The Python's Lair** — Face the final boss!

### How Combat Works
Each enemy asks you a Python question. Answer correctly to defeat them and earn XP, gold, and loot. Answer wrong and the enemy attacks — you lose HP. If your HP hits 0, you revive at camp and can try again. Spend gold at the inn to heal up.

## Running Locally

No server needed! Just open `docs/index.html` in a browser:

```bash
open docs/index.html
```

## Terminal Mode (Advanced)

A classic terminal quiz mode is also included for users who prefer the command line:

```bash
python3 main.py
```
